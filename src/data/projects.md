# Projects

## Biomedical Biomarker Analysis

- Domain: Biomedical machine learning, biomarker discovery, artificial intelligence for molecular data.
- Summary: Machine learning was conducted on the MiceProtein dataset using df-analyze to predict learning behavior in mice. The project identified six key cortical protein biomarkers associated with learning and Down syndrome-related analysis.
- Methods: Supervised machine learning, feature selection, redundancy-aware analysis, reproducible data workflows.
- GitHub: https://github.com/raymondstfx/DataCleaningForMiceProtein

## Effective Crowd Counting Using Deep Learning Networks

- Domain: Computer vision, crowd counting, efficient deep learning.
- Summary: Honours thesis work improved a lightweight crowd counting model based on MobileCount and C3-style frameworks. The project enhanced adaptability to varying crowd densities through multi-column dilated convolutions and improved feature fusion.
- Dataset/Evaluation: ShanghaiTech Part B dataset; evaluated with MAE and RMSE.
- GitHub: https://github.com/raymondstfx/Thesis
